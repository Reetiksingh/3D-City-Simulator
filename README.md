# 🏙️ Smart City Simulator (P5.js)

An interactive **Smart City Simulation** built using **p5.js**, where users can design, manage, and optimize a virtual city while balancing infrastructure, environment, and citizen well-being.

---

## 🚀 Project Overview

This project simulates a **grid-based smart city** where users can place different types of buildings, manage disasters, and receive AI-based suggestions to improve city planning.

It combines:

* 🧠 Logic building (DSA concepts like grids & adjacency)
* 🎨 Interactive UI (p5.js canvas)
* 🤖 Basic AI assistant (rule-based chatbot + advisor system)
* 🌦️ Environmental simulation (weather + disasters)

---

## 🎯 Key Features

### 🏗️ 1. Interactive City Builder

* 10x10 grid system
* Place buildings like:

  * 🏠 Home
  * 🏫 School
  * 🏥 Hospital
  * 🏭 Factory
  * 🌳 Park
  * 🛍️ Mall
  * 👮 Police
  * 🛕 Temple
  * 🛣️ Roads
* Real-time visual feedback using emojis

---

### ⚡ 2. Smart Scoring System

* Score is calculated based on **adjacency rules**
* Examples:

  * Homes near parks → ✅ Positive score
  * Factories near hospitals → ❌ Negative score
  * Roads connecting buildings → ➕ Boost score

👉 Demonstrates **graph-like thinking & spatial logic**

---

### 🤖 3. AI Advisor System

* Analyzes your city layout
* Suggests improvements like:

  * “Move factory away from school”
  * “Add hospital near homes”
  * “Improve road connectivity”

👉 Simulates **decision-support AI**

---

### 🌪️ 4. Disaster Simulation

* Random fire events 🔥
* Encourages planning for emergency response
* Adds realism + dynamic gameplay

---

### 🌦️ 5. Weather System

* Dynamic weather conditions:

  * ☀️ Sunny
  * 🌧️ Rainy
  * ❄️ Snowy
  * ⛈️ Stormy
* Weather impacts scoring logic

---

### 💾 6. Save & Load System

* Uses **localStorage**
* Persist city layouts
* Enables scenario testing

---

### 🔁 7. Undo / Redo Functionality

* Stack-based state management
* Improves UX and debugging

---

### 🤖 8. IoT Chatbot Assistant

* Built-in chatbot UI
* Answers predefined smart city queries
* Example:

  * “How to reset device?”
  * “Battery status?”
  * “Factory emissions?”

👉 Represents **IoT + Smart Assistant integration**

---

### 🚗 9. Autonomous Robot Simulation

* Moving robot 🤖 across grid
* Toggle ON/OFF
* Represents:

  * Surveillance
  * Smart mobility
  * Automation

---

### 🎯 10. Auto City Generator

* “Generate Perfect City” feature
* Creates optimized layout using distance logic
* Demonstrates:

  * Algorithmic thinking
  * Heuristic design
